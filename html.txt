<html> - Starts the HTML document
<body> - Where all visible content goes
</body> - Closes the body section
</html> - Ends the HTML document

<!DOCTYPE html>
<html>
<head>
  <title>My First Page</title>
</head>
<body>
  This is my very first webpage!
</body>
</html>

<!DOCTYPE html> - Tells the browser this is an HTML5 document
<head> - Contains information about the page (not visible)
<title> - The text that appears in the browser tab



<!DOCTYPE html>
<html>
<head>
  <title>My Simple Page</title>
</head>
<body>
  <h1>Welcome to My Website</h1>
  <p>This is my first paragraph.</p>
  <p>This is my second paragraph.</p>
</body>
</html>

<h1> - Main heading (biggest text)
<p> - Paragraph


<!DOCTYPE html>
<html>
<head>
  <title>My Page with Picture</title>
</head>
<body>
  <h1>My Favorite Animal</h1>
  <p>Here's a picture of a cute dog:</p>
  <img src="https://images.unsplash.com/photo-1518717758536-85ae29035b6d" width="300">
</body>
</html>

<img> - Inserts an image (notice: no closing tag needed!)
src - Where the image comes from (URL or file name)
width - How wide the image should be (in pixels)


<!DOCTYPE html>
<html>
<head>
  <title>My Page with Link</title>
</head>
<body>
  <h1>Welcome!</h1>
  <p>Click here to visit Google:</p>
  <a href="https://www.google.com">Go to Google</a>
  
  <p>Here's that cute dog again:</p>
  <img src="https://images.unsplash.com/photo-1518717758536-85ae29035b6d" width="200">
</body>
</html>

<a> - Creates a clickable link (anchor)
href - Where the link goes when clicked
The text between <a> and </a> is what you click on

<ul> - Unordered list (bullets)
<li> - List item (each bullet point)
